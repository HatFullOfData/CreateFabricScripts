# CreateFabricScripts

## Overview

This repository contains scripts designed to create and populate demo data for Power BI training scenarios that utilize Microsoft Fabric as the data storage platform. The dataset is based on a fictional business model: a company that supplies party products to retail shops and event organizers.

## Purpose

The primary purpose of this repository is to provide a realistic, comprehensive dataset that can be used for:
- **Power BI Training**: Teaching data modeling, report creation, and dashboard development
- **Microsoft Fabric Demonstrations**: Showcasing Fabric's capabilities for data storage and management
- **Business Intelligence Practice**: Offering a practical scenario for learning BI concepts

## Business Scenario

The demo data represents a party supply company with the following characteristics:

- **Product Range**: Party supplies including decorations, tableware, balloons, costumes, and party favors
- **Customer Base**: 
  - Retail shops that stock party supplies
  - Event organizers planning parties, weddings, and corporate events
- **Business Operations**: Sales transactions, inventory management, supplier relationships, and customer orders

This business model provides a rich dataset with multiple dimensions and facts, making it ideal for creating complex Power BI reports and learning advanced analytics techniques.

## Getting Started

The scripts in this repository will help you set up the demo environment and populate it with realistic data that mirrors common business scenarios in the party supply industry.

## Technology Stack

- **Microsoft Fabric**: Cloud-based data platform for storage and analytics
- **Power BI**: Business intelligence and data visualization tool
